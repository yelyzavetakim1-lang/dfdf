How to use
1. Replace your current index.html with this new index.html
2. Replace your current styles.css / styles.css equivalent with styles.css from this package
3. Add script.js to the repository root
4. Add the en folder to the repository root
5. Keep apple-touch-icon.png, favicon.svg and site.webmanifest in the root
6. Commit and push to GitHub Pages
